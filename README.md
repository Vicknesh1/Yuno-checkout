# Yuno Full SDK Checkout Demo

A minimal, production-ready implementation of Yuno's Full SDK for embedded credit card payments.

 Features

✓ SDK FULL- Embedded checkout (no redirects)
✓ Credit Card - Via Yuno Testing Gateway
✓ Flexible Design - Easy to add new payment methods via Dashboard
✓ End-to-end Flow - Dashboard routing → Backend API → Frontend checkout

 Tech Stack

Frontend: HTML, JavaScript, Yuno Full SDK
Backend: Node.js, Express
Testing: Yuno Testing Gateway

Project Structure : yuno-demo/
├── frontend/ # Static HTML + JS checkout page
├── backend/ # Express API server
├── README.md
